# 🧪 لوحة اختبارات SHAMEL

> [!tip] طريقة إضافة مشكلة جديدة
> اضغط على القالب `Templates/Bug-Template.md` أو انسخ ملف موجود في `Issues/`، غيّر العنوان والـ frontmatter، واحفظ. اللوحة تتحدث تلقائياً.

## 📊 ملخص سريع

```dataview
TABLE WITHOUT ID
  status AS "الحالة",
  length(rows) AS "العدد"
FROM "Testing-Logs/Issues"
WHERE type = "bug"
GROUP BY status
SORT length(rows) DESC
```

## 🔴 المشاكل المفتوحة (حسب الخطورة)

```dataview
TABLE WITHOUT ID
  ("[[" + file.name + "|" + id + "]]") AS "ID",
  file.link AS "العنوان",
  severity AS "الخطورة",
  role AS "الدور",
  platform AS "المنصة",
  area AS "الشاشة",
  date_found AS "تاريخ الاكتشاف"
FROM "Testing-Logs/Issues"
WHERE type = "bug" AND status != "fixed" AND status != "wont-fix"
SORT choice(severity = "critical", 0, choice(severity = "high", 1, choice(severity = "medium", 2, 3))) ASC
```

## 🟢 تم إصلاحها

```dataview
TABLE WITHOUT ID
  file.link AS "العنوان",
  severity AS "الخطورة",
  date_fixed AS "تاريخ الإصلاح"
FROM "Testing-Logs/Issues"
WHERE type = "bug" AND status = "fixed"
SORT date_fixed DESC
```

## 🧩 تجميع حسب المنصة

```dataview
TABLE WITHOUT ID
  platform AS "المنصة",
  length(filter(rows, (r) => r.status != "fixed")) AS "مفتوحة",
  length(rows) AS "الكل"
FROM "Testing-Logs/Issues"
WHERE type = "bug"
GROUP BY platform
```

## 👥 تجميع حسب الدور

```dataview
TABLE WITHOUT ID
  role AS "الدور",
  length(filter(rows, (r) => r.status != "fixed")) AS "مفتوحة"
FROM "Testing-Logs/Issues"
WHERE type = "bug"
GROUP BY role
```

## 🤖 طلب الذكاء الاصطناعي حل المشاكل المفتوحة

انسخ هذا الاستعلام ناتجه (قائمة المشاكل المفتوحة) والصقه للذكاء الاصطناعي، أو اطلب منه:
> "اقرأ المشاكل المفتوحة في `Testing-Logs/Issues` (status != fixed) وحلها واحدة واحدة."

```dataview
LIST WITHOUT ID
  (id + " — " + file.link + " (" + severity + " / " + platform + ")")
FROM "Testing-Logs/Issues"
WHERE type = "bug" AND status != "fixed" AND status != "wont-fix"
SORT severity ASC
```
